# Andreas Maler — Dungeon World Character Sheet

A fully interactive Dungeon World character sheet for **Andreas Maler**, Cleric of the Order of the Kindled Breath, follower of Dolores Dei.

Built with React. Inspired by [Disco Elysium](https://discoelysium.com).

## Features

- **Character Tab** — Stats, HP, armor, XP, bonds, portrait
- **Spells Tab** — Full Cleric spell system with preparation tracking, ongoing penalties, commune reset
- **Moves Tab** — All basic, special, and class moves with tap-to-roll dice
- **Gear Tab** — Inventory with load tracking
- **Lore Tab** — Complete backstory of Andreas and the Dolorian faith
- **World Tab** — Illustrated gallery of artifacts, icons, and key figures
- **Reference Tab** — Quick-reference cards for at-the-table play
- **Log Tab** — Roll history

## How to Use

Just open `index.html` in any browser. All data saves to your browser's localStorage automatically.

### Sync Between Devices

Use the **Sync** button to generate a 6-character code, then enter it on your other device. Or use the manual JSON export/import for full backups.
